# React Not Uygulaması
React.js ile geliştirdiğim not tutma uygulaması.Kayıt edilen notlar localStorage aracılığı ile tarayıcıda saklanabilmektedir.

## Kullandığım Teknolojiler
* React.js
* Pure CSS
* Context API
* localStorage
